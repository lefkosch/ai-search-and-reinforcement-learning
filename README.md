# AI Search and Reinforcement Learning

A collection of artificial intelligence projects developed as part of coursework at Harokopio University of Athens.

## Overview

This repository includes implementations of fundamental AI algorithms used in search problems, decision-making, and reinforcement learning.

## Projects

### Search Algorithms & Multi-Agent Systems

* Depth-First Search (DFS)
* Breadth-First Search (BFS)
* Uniform Cost Search (UCS)
* A* Search
* Minimax
* Alpha-Beta Pruning
* Expectimax

### Reinforcement Learning

* Value Iteration
* Q-Learning
* Epsilon-greedy strategies

## Technologies

* Python

## My Contribution

* Implemented and modified core search algorithms (DFS, BFS, UCS, A*)
* Developed multi-agent decision algorithms (Minimax, Alpha-Beta, Expectimax)
* Implemented reinforcement learning algorithms (Value Iteration, Q-Learning)

## Notes

This project was developed as part of academic coursework. The base structure was provided, and the main focus was on implementing and modifying algorithmic logic.
